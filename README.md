# HabitForge

HabitForge is a full-stack habit tracking web application designed to help users build consistency through daily tracking, streak monitoring, and progress visualization.

## Live Demo
https://habitforge-gamma.vercel.app/

## Features
- User authentication
- Create and manage habits
- Habit categories
- Progress tracking
- Streak analytics
- Interactive dashboard
- Heatmap visualization

## Tech Stack
- React
- TypeScript
- Vite
- Supabase
- Tailwind CSS
- TanStack Router

## Installation

```bash
git clone https://github.com/Numinexora/habitforge.git
cd habitforge
npm install
npm run dev
```

## Deployment
Deployed on Vercel.
